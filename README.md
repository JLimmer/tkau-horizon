# Te Rua Horizon Command Center

**Te Kura Te Au — Strategic Project Management App**

A Progressive Web App (PWA) for managing the TKAU Te Rua Bay project.

## Sections
- Dashboard — Net position, ridge line analysis, jetty countdown
- Capital Ledger — Transaction tracking, asset values
- Budget Planner — 160+ line item stress test with pricing
- Risk Matrix — Liability and compliance tracking
- Site Specifications — Infrastructure and consent registry
- The Blitz — Biarra Sprint task management

## Install on Android
1. Open Chrome on your Android phone
2. Navigate to: **https://jlimmer.github.io/tkau-horizon/**
3. Tap the menu (⋮) → "Add to Home screen"
4. Tap "Add" — the app installs like a native app

## Files
- `index.html` — Main application
- `manifest.json` — PWA manifest
- `sw.js` — Service worker (enables offline use)
- `icon-192.png` / `icon-512.png` — App icons
